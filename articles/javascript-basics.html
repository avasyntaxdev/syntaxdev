تعلم JavaScript من الصفر للمبتدئين (دليل شامل)

إذا كنت تريد دخول عالم تطوير الويب فلابد من تعلم JavaScript، فهي لغة البرمجة الأساسية التي تجعل صفحات الويب تفاعلية. في هذا الدليل سنبدأ من الصفر تماماً خطوة بخطوة.

ما هي JavaScript ؟

JavaScript هي لغة برمجة تعمل داخل المتصفح، وتستخدم لإضافة التفاعل مثل الأزرار والنماذج والتنبيهات وتحديث المحتوى بدون إعادة تحميل الصفحة.

أول مثال في JavaScript


console.log("Hello World");

يتم عرض النص داخل وحدة التحكم (Console) في المتصفح.

المتغيرات في JavaScript


let name = "Ali";
const age = 25;

- نستخدم let للقيم المتغيرة - نستخدم const للقيم الثابتة

الدوال (Functions)


function greet(){
  console.log("مرحبا بك");
}

greet();

الدوال تساعدك على إعادة استخدام الكود وتنظيمه.

التعامل مع DOM

DOM هو تمثيل عناصر الصفحة داخل المتصفح. يمكنك التحكم به باستخدام JavaScript. 


document.getElementById("btn").onclick = function(){
  alert("تم الضغط");
}

الكود أعلاه يجعل الزر يعرض رسالة عند الضغط عليه.

لماذا يجب تعلم JavaScript؟

  • مطلوبة في جميع وظائف تطوير الويب
  • أساس مكتبات مثل React و Vue
  • تستخدم في الواجهة الأمامية والخلفية (Node.js)
  • لغة سهلة البداية وقوية جداً

أسئلة شائعة حول تعلم JavaScript

لا، يمكن تعلم الأساسيات خلال أسابيع قليلة مع التطبيق العملي.
