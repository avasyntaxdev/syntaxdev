شرح React Hooks للمبتدئين (useState و useEffect)

تعتبر React Hooks من أهم المميزات الحديثة في React، حيث تسمح لك باستخدام الحالة (State) ودورة حياة المكون بدون الحاجة إلى Class Components.

ما هي React Hooks ؟

React Hooks هي دوال خاصة تم إضافتها في React 16.8 وتسمح لك باستخدام ميزات مثل State و Lifecycle داخل Functional Components.

أولاً: شرح useState

Hook useState تُستخدم لإنشاء حالة (State) داخل المكون.

مثال عملي:


import { useState } from "react";

function Counter() {

  const [count, setCount] = useState(0);

  return (
    

      
{count}

      
    

  );
}

شرح الكود:

  • count → القيمة الحالية
  • setCount → دالة تحديث القيمة
  • useState(0) → القيمة الابتدائية

ثانياً: شرح useEffect

Hook useEffect تُستخدم لتنفيذ كود بعد تحميل المكون أو عند تغيير قيمة معينة.

مثال:


import { useEffect } from "react";

useEffect(() => {
  console.log("Component Loaded");
}, []);

ماذا يعني [] ؟

  • [] → يعمل مرة واحدة عند تحميل المكون
  • [count] → يعمل عند تغير count
  • بدون مصفوفة → يعمل عند كل إعادة رسم

مثال يجمع useState و useEffect


import { useState, useEffect } from "react";

function App() {

  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `Count: ${count}`;
  }, [count]);

  return (
    
  );
}

أهم قواعد استخدام Hooks

  • يجب استدعاء Hooks في أعلى المكون
  • لا تستخدم Hooks داخل شروط أو حلقات
  • استخدمها فقط داخل Functional Components

أسئلة شائعة حول React Hooks

هل Hooks أفضل من Class Components؟

نعم، لأنها أبسط وأسهل في القراءة وإعادة الاستخدام.

هل يمكن استخدام أكثر من useState؟

نعم، يمكنك استخدام أكثر من Hook داخل نفس المكون.
