شرح SQL للمبتدئين (تعلم من الصفر)

إذا كنت تريد دخول عالم البرمجة أو تطوير المواقع، فإن تعلم SQL أمر أساسي للتعامل مع قواعد البيانات. في هذا الدليل سنتعلم أهم أوامر SQL خطوة بخطوة.

ما هي SQL ؟

SQL اختصار لـ Structured Query Language، وهي لغة تستخدم للتعامل مع قواعد البيانات مثل MySQL و PostgreSQL.

أمر SELECT (جلب البيانات)

يستخدم لجلب البيانات من جدول:


SELECT * FROM users;

جلب أعمدة محددة:


SELECT name, email FROM users;

استخدام شرط WHERE:


SELECT * FROM users WHERE id = 1;

أمر INSERT (إضافة بيانات)


INSERT INTO users(name, email)
VALUES('Ali', 'ali@mail.com');

أمر UPDATE (تعديل بيانات)


UPDATE users
SET name = 'Ahmed'
WHERE id = 1;

أمر DELETE (حذف بيانات)


DELETE FROM users
WHERE id = 1;

استخدام ORDER BY و LIMIT


SELECT * FROM users
ORDER BY id DESC
LIMIT 5;

إنشاء جدول جديد


CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100)
);

أفضل ممارسات عند استخدام SQL

  • استخدم WHERE لتجنب تعديل كل البيانات
  • لا تستخدم DELETE بدون شرط
  • احمِ تطبيقك من SQL Injection
  • استخدم الفهارس (Indexes) لتحسين الأداء

أسئلة شائعة حول SQL

هل SQL صعبة للمبتدئين؟

لا، فهي لغة بسيطة وتعتمد على أوامر واضحة.

ما الفرق بين SQL و MySQL؟

SQL هي لغة، أما MySQL فهو نظام إدارة قواعد بيانات يستخدم SQL.
