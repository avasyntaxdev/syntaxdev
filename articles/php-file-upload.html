رفع الملفات في PHP خطوة بخطوة (File Upload Tutorial)

يعد رفع الملفات في PHP من أهم المهارات لأي مطور ويب، سواء كنت تبني نظام تسجيل مستخدمين أو لوحة تحكم. في هذا الدليل سنتعلم الطريقة الصحيحة مع الحماية من الأخطاء الشائعة.

أولاً: إنشاء نموذج رفع ملف (Form)

يجب استخدام enctype="multipart/form-data" داخل الفورم:

ثانياً: استقبال الملف في PHP

شرح مصفوفة $_FILES

  • name : اسم الملف
  • tmp_name : المسار المؤقت
  • size : حجم الملف
  • type : نوع الملف
  • error : كود الخطأ إن وجد

التحقق من نوع الملف (مهم جداً)


$allowed = ['jpg','png','jpeg'];
$extension = pathinfo($fileName, PATHINFO_EXTENSION);

if(in_array($extension, $allowed)){
   move_uploaded_file($tmpName, "uploads/" . $fileName);
}else{
   echo "نوع الملف غير مسموح";
}

تحديد حجم الملف الأقصى


if($_FILES['file']['size'] > 2000000){
   echo "حجم الملف كبير جداً";
}

أفضل ممارسات أمان عند رفع الملفات

  • لا تسمح برفع ملفات PHP
  • غيّر اسم الملف لتجنب التكرار
  • تحقق من الامتداد والحجم
  • احفظ الملفات خارج public إن أمكن

أشهر مشاكل رفع الملفات في PHP

1- الملف لا يتم رفعه

تأكد من:
  • وجود مجلد uploads
  • إعطاء صلاحية 755 أو 775
  • تعديل upload_max_filesize في php.ini

2- خطأ move_uploaded_file

غالباً بسبب مسار خاطئ أو صلاحيات غير كافية.
