إنشاء REST API باستخدام Node.js و Express (دليل عملي)

إذا كنت تريد بناء Backend حديث فلابد من تعلم إنشاء REST API باستخدام Node.js و Express. في هذا الدليل سنبدأ من الصفر حتى تشغيل API بسيط.

ما هو REST API ؟

REST API هو أسلوب لبناء خدمات ويب تعتمد على بروتوكول HTTP وتستخدم Methods مثل GET و POST و PUT و DELETE للتعامل مع البيانات.

الخطوة 1: إنشاء مشروع Node.js


npm init -y

الخطوة 2: تثبيت Express


npm install express

الخطوة 3: إنشاء ملف index.js


const express = require("express");
const app = express();

app.use(express.json());

app.get("/", (req, res) => {
  res.send("Hello API");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});

الآن افتح المتصفح وادخل: 


http://localhost:3000

إنشاء Route باستخدام GET


app.get("/users", (req, res) => {
  res.json([
    { id: 1, name: "Ali" },
    { id: 2, name: "Sara" }
  ]);
});

إنشاء Route باستخدام POST


app.post("/users", (req, res) => {
  const user = req.body;
  res.json({
    message: "User created",
    data: user
  });
});

اختبار API باستخدام Postman

يمكنك استخدام أداة Postman لإرسال طلبات GET و POST واختبار الـ API بسهولة.

أفضل ممارسات بناء REST API

  • استخدم express.json() لقراءة البيانات
  • افصل Routes في ملفات مستقلة عند كبر المشروع
  • استخدم Status Codes مناسبة (200 - 404 - 500)
  • تحقق من صحة البيانات قبل الحفظ

أسئلة شائعة حول Node و Express

هل Express مناسب للمشاريع الكبيرة؟

نعم، لكنه يحتاج تنظيم جيد وفصل الملفات.

ما الفرق بين REST و GraphQL؟

REST يعتمد على عدة Endpoints، بينما GraphQL يستخدم Endpoint واحد ويحدد البيانات المطلوبة.

هل Node.js سريع في بناء APIs؟

نعم، Node.js معروف بأدائه الجيد خاصة في التطبيقات ذات الطلبات المتزامنة.
