شرح أهم ميزات ES6 في JavaScript (مع أمثلة عملية)

إصدار ES6 (المعروف أيضاً باسم ECMAScript 2015) قدم تحديثات قوية جعلت كتابة كود JavaScript أسهل وأكثر تنظيماً. في هذا المقال سنشرح أهم الميزات مع أمثلة بسيطة.

1- Arrow Functions

توفر طريقة مختصرة لكتابة الدوال. 


const sum = (a, b) => a + b;

تتميز Arrow Functions بأنها لا تنشئ this خاص بها، بل ترثه من النطاق الخارجي.

2- let و const

في ES6 تم تقديم كلمتي let و const بدلاً من var. 


let name = "Ali";
const age = 25;
  • let تستخدم للقيم التي يمكن تغييرها
  • const تستخدم للقيم الثابتة

3- Template Literals

تسمح بكتابة نصوص متعددة الأسطر وإدخال المتغيرات داخل النص. 


let name = "Ali";
console.log(`مرحبا ${name}`);

4- Destructuring

تسمح باستخراج القيم من المصفوفات أو الكائنات بسهولة. 


const user = {name: "Ali", age: 25};
const {name, age} = user;

5- Default Parameters


function greet(name = "Guest"){
  console.log(name);
}

لماذا تعتبر ES6 مهمة؟

  • تنظيم أفضل للكود
  • اختصار في الكتابة
  • سهولة قراءة الكود
  • تعتمد عليها مكتبات مثل React

هل يجب تعلم ES6 قبل React؟

ما الفرق بين var و let؟

let تحترم نطاق الكتلة (Block Scope) بينما var لا تفعل ذلك.
