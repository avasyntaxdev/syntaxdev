تعلم Git و GitHub للمبتدئين (دليل عملي كامل)

إذا كنت تريد إدارة مشاريعك البرمجية باحترافية فلابد من تعلم Git و GitHub. في هذا الدليل سنشرح الأساسيات خطوة بخطوة بطريقة بسيطة وعملية.

ما هو Git ؟

Git هو نظام لإدارة الإصدارات (Version Control System) يسمح لك بحفظ التعديلات على مشروعك والرجوع لأي نسخة سابقة.

ما هو GitHub ؟

GitHub هو منصة لاستضافة مشاريع Git على الإنترنت ومشاركة الكود مع الآخرين.

أهم أوامر Git الأساسية

1- إنشاء مستودع جديد


git init

2- إضافة الملفات للتتبع


git add .

3- حفظ التعديلات (Commit)


git commit -m "first commit"

4- معرفة حالة المشروع


git status

5- مشاهدة السجل


git log

طريقة رفع مشروعك إلى GitHub

1- إنشاء Repository جديد في GitHub

قم بإنشاء مستودع جديد من حسابك.

2- ربط المشروع المحلي بالمستودع


git remote add origin URL

3- رفع المشروع


git push -u origin main

أخطاء شائعة قد تواجهك

  • رفض الرفع بسبب عدم وجود commit
  • خطأ في اسم الفرع (main / master)
  • مشاكل في المصادقة (Authentication)

أسئلة شائعة حول Git

هل Git صعب للمبتدئين؟

لا، الأساسيات بسيطة جداً ويمكن تعلمها خلال يوم واحد.

ما الفرق بين Git و GitHub؟

Git أداة محلية لإدارة الإصدارات، أما GitHub فهو منصة استضافة.

هل يمكن استخدام Git بدون GitHub؟

نعم، يمكنك استخدام Git محلياً بدون رفع المشروع إلى الإنترنت.
